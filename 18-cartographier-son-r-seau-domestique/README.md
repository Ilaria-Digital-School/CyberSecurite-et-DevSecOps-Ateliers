# Cartographier son réseau domestique

**Rappel théorique** Avant de commencer cet atelier, revenons sur les concepts fondamentaux abordés dans les cours précédents. Un réseau informatique est un ensemble d'hôtes (ordinateurs, smartphones, imprimantes, box internet, objets connectés) reliés entre eux par des liens (filaires via câble Ethernet, ou sans fil via Wi-Fi). La communication entre ces hôtes s'effectue par l'échange de paquets, c'est-à-dire des blocs de données structurés transportant à la fois les données utiles et les informations d'adressage (adresse IP source, adresse IP destination). Chaque hôte sur un réseau possède au minimum une adresse IP (ex. : 192.168.1.42) qui l'identifie de manière unique sur ce réseau local, et une adresse MAC qui identifie physiquement sa carte réseau. Votre box internet joue le rôle de passerelle (gateway) : c'est elle qui relie votre réseau domestique à Internet. Elle attribue généralement les adresses IP locales via un mécanisme appelé DHCP. Erreur fréquente des débutants : confondre l'adresse IP publique (celle vue depuis Internet) et l'adresse IP privée (celle utilisée en interne, typiquement en 192.168.x.x ou 10.x.x.x). Une autre erreur courante est de croire que seuls les ordinateurs sont des hôtes : une ampoule connectée, une Smart TV ou une console de jeu sont également des hôtes sur votre réseau. Mini-exemple : chez vous, votre box (192.168.1.1) distribue des adresses à votre PC (192.168.1.10), votre téléphone (192.168.1.11) et votre télévision (192.168.1.20).
